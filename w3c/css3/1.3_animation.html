动画-animation
  • animation 主要为元素设置动画，其为一个复合属性，包括：
    1、animation-name 指定动画的名字，通过@keyframes定义。
    2、animation-duration 动画的持续时间。
    3、animation-delay 延迟时间。
    4、animation-iteration-count 播放次数，无限次为:infinite
    5、animation-timing-function 变形函数ease、linear、ease-in、ease-out、ease-in-out。
    6、animation-direction 播放方向：normal播放完成结束；alternate:偶数次反方向回到初始状态。
    7、animation-play-state 播放状态:paused和running用于暂停和运行。
    复合属性也可写为animation:name dur delay count fun dir
  • keyframes 关键帧，用于定义动画：
    例如：
			@keyframes  rotatefrm{
				0%{
					-webkit-transform:rotate(0deg);
					-ms-transform:rotate(0deg);
					-moz-transform:rotate(0deg);
					transform:rotate(0deg);
				}
				50%{
					background:#ffffff;
					-webkit-transform:rotate(180deg);
					-ms-transform:rotate(180deg);
					-moz-transform:rotate(180deg);
					transform:rotate(180deg);
				}
				100%{
					-webkit-transform:rotate(3590deg);
					-ms-transform:rotate(359deg);
					-moz-transform:rotate(359deg);
					transform:rotate(359deg);
				}
			}
